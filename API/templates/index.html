Documentation:

Database Access:

To access the database, make requests to http://138.197.49.155:8000/api/database

Required parameters: query, key

Request Example: /Scripts/testrequest.py

#testrequest.py
import requests

#make request to API
r = requests.post('http://138.197.49.155:8000/api/database/', data = {'query': 'select * from users', 'key': 'password is in test file'})
print(r.json())

The database was created with

CREATE TABLE users (id integer, name varchar(50), tracked varchar(9999), untracked varchar(9999), goals varchar(9999), themes varchar(9999), competition varchar(9999), inProgressWorkouts varchar(9999), savedWorkouts varchar(9999));

The varchar(9999) fields are meant to hold JSON objects as strings.
